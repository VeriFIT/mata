48/**
49 * Creates Tabakov-Vardi random NFA.
50 * The implementation is based on the paper "Experimental Evaluation of Classical Automata Constructions" by Tabakov and Vardi.
51 *
52 * @param num_of_states Number of states in the automaton.
53 * @param alphabet_size Size of the alphabet.
54 * @param states_transitions_ratio_per_symbol Ratio between number of transitions and number of states for each symbol.
55 * The value must be in range [0, num_of_states]. A value of 1 means that there will be num_of_states transitions for each symbol.
56 * A value of num_of_states means that there will be a transition between every pair of states for each symbol.
57 * @param final_state_density Density of final states in the automaton. The value must be in range [0, 1]. The state 0 is always final.
58 * If the density is 1, every state will be final.
59 */
60Nfa create_random_nfa_tabakov_vardi(const size_t num_of_states, const size_t alphabet_size, const double states_trans_ratio_per_symbol, const double final_state_density);

108/**
109 * @brief Create NFA from @p regex
110 *
111 * The created NFA does not contain epsilons, is trimmed and reduced.
112 * It uses the parser from RE2, see mata/parser/re2praser.hh for more
113 * details and options.
114 *
115 * At https://github.com/google/re2/wiki/Syntax, you can find the syntax
116 * of @p regex with following futher limitations:
117 * 1) The allowed characters are the first 256 characters of Unicode,
118 * i.e., Latin1 encoding (ASCII + 128 more characters). For the
119 * full Unicode, check mata/parser/re2praser.hh.
120 * 2) The created automaton represents the language of the regex and
121 * is not expected to be used in regex matching. Therefore, stuff
122 * like ^, $, \b, etc. are ignored in the regex. For example, the
123 * regular expressions a*b and ^a*b will both result in the same
124 * NFA accepting the language of multiple 'a's followed by one 'b'.
125 * See also issue #494.
126 *
127 * @sa mata::parser::create_nfa()
128 *
129 * @param regex regular expression
130 */
131Nfa create_from_regex(const std::string& regex);
