296/**
297 * @brief Delta is a data structure for representing transition relation.
298 *
299 * Transition is represented as a triple Trans(source state, symbol, target state). Move is the part (symbol, target
300 * state), specified for a single source state.
301 * Its underlying data structure is vector of StatePost classes. Each index to the vector corresponds to one source
302 * state, that is, a number for a certain state is an index to the vector of state posts.
303 * Transition relation (delta) in Mata stores a set of transitions in a four-level hierarchical structure:
304 * Delta, StatePost, SymbolPost, and a set of target states.
305 * A vector of 'StatePost's indexed by a source states on top, where the StatePost for a state 'q' (whose number is
306 * 'q' and it is the index to the vector of 'StatePost's) stores a set of 'Move's from the source state 'q'.
307 * Namely, 'StatePost' has a vector of 'SymbolPost's, where each 'SymbolPost' stores a symbol 'a' and a vector of
308 * target states of 'a'-moves from state 'q'. 'SymbolPost's are ordered by the symbol, target states are ordered by
309 * the state number.
310 */
311class Delta {
312public:
313 inline static const StatePost empty_state_post; // When posts[q] is not allocated, then delta[q] returns this.
314
315 Delta(): state_posts_{} {}

318 explicit Delta(size_t n): state_posts_{ n } {}

320 Delta& operator=(const Delta& other) = default;

323 bool operator==(const Delta& other) const;

329 /**
330 * @brief Get constant reference to the state post of @p source.
331 *
332 * If we try to access a state post of a @p source which is present in the automaton as an initial/final state,
333 * yet does not have allocated space in @c Delta, an @c empty_post is returned. Hence, the function has no side
334 * effects (no allocation is performed; iterators remain valid).
335 * @param source[in] Source state of a state post to access.
336 * @return State post of @p source.
337 */
338 const StatePost& state_post(const State source) const {

357 * @brief Get mutable (non-constant) reference to the state post of @p source.

363 * If we try to access a state post of a @p source which is present in the automaton as an initial/final state,

368 * to make sure that posts of all states in the automaton are allocated, e.g., write an NFA method that allocate
369 * @c Delta for all states of the NFA.
