160/**
161 * Enumerated alphabet using a set of integers as symbols maintaining a set of specified symbols.
162 *
163 * @c EnumAlphabet is a version of direct (identity) alphabet (does not give names to symbols, their name is their
164 * integer value directly). However, unlike @c IntAlphabet, @c EnumAlphabet maintains an ordered set of symbols in
165 * the alphabet.
166 *
167 * Therefore, calling member functions @c get_complement() and @c get_alphabet_symbols() makes sense in the context
168 * of @c EnumAlphabet and the functions give the expected results.
169 *
170 * Example:
171 * ```cpp
172 * Alphabet alph{ EnumAlphabet{ 0, 4, 6, 8, 9 } };
173 * CHECK(alph.translate_symb("6") == 6);
174 * CHECK_THROWS(alph.translate_symb("5")); // Throws an exception about an unknown symbol.
175 * CHECK(alph.get_complement({ utils::OrdVector<Symbol>{ 0, 6, 9 } }) == utils::OrdVector<Symbol>{ 4, 8 });
176 * ```
177 */
